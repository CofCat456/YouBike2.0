# YouBike 微笑單車

Welcome to YouBike 微笑單車, 推廣民眾騎乘自行車作為短程接駁交通工具，用慢速的方式體驗城市不同時空的迷人風情，不僅提供市民便利的接駁工具，同時還兼具了休閒運動與娛樂的特點，讓市民及遊客體驗騎乘自行車是一種生活與旅遊的最佳方式。

## :wrench: 技術

- NextJS 14
- App Router
- Middleware
- Ant Design 5.0
